# 空き家3Dリフォーム提案Webサービス 要件整理

## Audience

- 要件を短時間で共有したい関係者
- 自治体、不動産、所有者、リフォーム事業者
- 実装の優先順位を確認したい開発側

## Objective

要件定義書の全体像を、16:9 の会議用スライドで一読できる形に整理する。

3. 想定ユースケース
4. 機能要件と MVP

6. 構成案、拡張、成功指標

## Source Plan

- 主ソース: `webgl/要件定義書.md`
- 補助確認: `webgl/mvp-plan.md`

## Visual System

- 16:9 横長、余白広め
- ネイビー、セージ、ゴールド、クレイを軸にした落ち着いた配色
- タイトルは太め、本文は読みやすさ優先
- 写真や生成画像に頼らず、編集しやすい図版中心

## Editability Plan

- 重要テキストはすべて PowerPoint 上の編集可能テキストボックスで構成
- 図版は角丸カード、チップ、簡易フロー図で表現
- 配布物は `.pptx` を最終成果物とする
